# Product images

Keep product photos here, grouped by category:

```text
product-images/sets/
product-images/rings/
product-images/necklace/
product-images/earings/
```

In Excel, use paths relative to `inventory/products.xlsx` / `inventory/products.xlsm`:

```text
product-images/rings/ring-front.jpg
product-images/rings/ring-side.jpg
```

For SVG files, add a same-name PNG next to the SVG for Excel note previews:

```text
placeholder-ring.svg   # website image
placeholder-ring.png   # Excel note preview
```
